# Student Score Prediction using Machine Learning

## Overview
This project predicts students' exam scores based on academic and lifestyle factors using regression techniques.

## Dataset
Student Performance Factors dataset (Kaggle).

## Methods Used
- Exploratory Data Analysis (EDA)
- Data preprocessing and encoding
- Linear Regression
- Polynomial Regression (degree = 2)

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

## Results
The linear regression model achieved an R² score of 0.64 with low prediction error, indicating a strong relationship between study-related features and exam performance.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
